Web Advisor Theme Builder is a powerful custom Gutenberg extension designed to help developers and designers build modern, dynamic WordPress layouts.

== Description ==

Web Advisor Theme Builder is an advanced Gutenberg block extension that allows developers, agencies, and designers to create modern, dynamic WordPress layouts visually—without coding.

This plugin comes with multiple high-quality custom blocks including a Bootstrap-powered Hero Slider, Font Awesome Social Media Block, Pop-up Modal Banner, and a fully featured Owl Carousel with responsive controls. Each block is optimized for performance, accessibility, and mobile responsiveness.

Every block supports flexible customization options such as spacing controls, colors, backgrounds, typography, animations, and inner block nesting. Whether you're building landing pages, product promos, blog sections, or business websites, Web Advisor Theme Builder gives you the creative tools to design professional-grade layouts directly inside the block editor.

A premium upgrade is available for users who want enhanced layouts, animations, template packs, and advanced block settings.

== Installation ==

1. Upload the plugin folder `web-advisor-theme-builder` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Open the block editor and search for “Web Advisor Blocks.”
4. Insert any block and customize it using the editor sidebar controls.

== Frequently Asked Questions ==

= Does this work with classic editor? =
No. This plugin is designed exclusively for the Gutenberg (Block Editor).

= Is a Pro version available? =
Yes, advanced sections, animation controls, and premium sliders will be available as paid upgrades.

= Does it slow down my site? =
No. All blocks are optimized for performance and load only the assets they need.

= Can I use this with any theme? =
Yes, it works with all modern block-compatible themes.
